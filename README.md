# SME Financial Analysis Tool

A free, instant web tool that analyses SME business financials 
the way a senior Relationship Manager reads a credit file.

## What it does

Calculates 9 key financial ratios across 3 years — 2 audited 
and 1 provisional — with instant colour-coded results, red flag 
detection, and a plain-English credit verdict.

Ratios covered: EBITDA Margin, Current Ratio, TOL/TNW, 
TOL/ATNW, DSCR, Debtor Days, Inventory Days, Creditor Days, 
and Interest Cover.

## Two views

- **RM View** — full ratio analysis, 3-year trend table, 
  red flag alerts, and a senior RM assessment note
- **Owner View** — clean summary in plain language

## Why I built this

 
I read hundreds of financial statements and saw the same 
mistakes repeatedly — this tool encodes that judgment.
